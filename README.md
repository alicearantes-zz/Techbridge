# Techbridge

Sobre o projeto:

O objetivo do sistema é realizar um diagnóstico digital de pequenas e médias empresas, usando metodologias ágeis para corrigir problemas internos nas mesmas, integrando Python com banco de dados MySQL. 

Tecnologias utilizadas

* Python
* MySQL
* MySQL Connector
* Brmodelo
* GitHub

Funcionalidades

* Cadastro de empresas
* Armazenamento de dados em banco de dados
* Realização de diagnósticos digitais
* Consulta e gerenciamento das informações cadastradas
